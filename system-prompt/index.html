Bloque 01 — Identidad y rol
System Prompt
Por qué esta arquitectura supera una lista de reglas
Identidad antes que instrucciones
Un modelo que sabe quién es adapta su comportamiento naturalmente. Las reglas se rompen; la identidad se mantiene.
Sesgos cognitivos distribuidos
Cada sesgo está ubicado donde genera mayor impacto. No son trucos — son principios de comportamiento humano aplicados al momento correcto.
Lógica de cierre explícita
Sin instrucciones de cierre, el modelo informa pero no vende. El prompt define exactamente cuándo y cómo pedir el dato que convierte.
